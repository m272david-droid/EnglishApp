# Dashboard UX Skill

Use this skill when designing or improving dashboard screens.

## Goal
Create a dashboard that motivates students to continue learning vocabulary daily.

The dashboard should feel like a modern EdTech product.

Target audience: teenagers aged 15–18.

---

## Design Principles

1. Clear visual hierarchy
2. One strong primary action
3. Visible progress toward a long-term goal
4. Clean spacing and readable cards
5. Mobile-first layout

---

## Structure Recommendation

1. Logo + title
2. Daily progress card
3. Start session CTA
4. Stats row (accuracy / streak / weekly progress)
5. Vocabulary progress map
6. Future features section

---

## Visual Style

Use:
- soft gradients
- layered cards
- subtle depth
- calm colors

Avoid:
- generic admin dashboards
- flat grid layouts
- overwhelming statistics
