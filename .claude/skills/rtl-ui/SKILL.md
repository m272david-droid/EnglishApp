# RTL UI Skill

Use this skill whenever editing UI in this project.

The application is Hebrew-first and must support RTL layouts correctly.

---

## RTL Rules

1. Layout should start from the right
2. Icons should respect RTL direction
3. Borders should use logical positioning
4. Text alignment should feel natural in Hebrew
5. Mobile layouts must remain readable

---

## Testing

Always check:
- card alignment
- button placement
- progress bars
- stats layout
